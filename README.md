# Vision for Cleaning Website

A premium, responsive website for Vision for Cleaning, built with Vite, React, and Tailwind CSS.

## üöÄ Getting Started

### Prerequisites
- **Node.js**: Required to run the project locally. (Download from [nodejs.org](https://nodejs.org/)))

### Installation
1.  Clone the repository.
2.  Install dependencies:
    ```bash
    npm install
    ```
3.  Start the development server:
    ```bash
    npm run dev
    ```

### Deployment (Vercel)
1.  Push this code to your GitHub repository.
2.  Import the repository in Vercel.
3.  Vercel will automatically detect the Vite settings.
4.  Click **Deploy**.
